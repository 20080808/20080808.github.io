手把手教hexo+github搭建个人博客

2017-08-28

趁着手热，就整理一下关于hexo整合github服务器上个人博客的搭建。
首先，先来了解下hexo。Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
详情文档 |Hexo

1.安装前提

安装 Hexo 相当简单。然而在安装前，您必须检查电脑中是否已安装下列应用程序：

  • Node.js

  • Git


    • 接下来只需要使用 npm 即可完成 Hexo 的安装。

    建站

    安装Hexo完成后，请执行下列命令，Hexo将会在指定文件夹中新建所需要的文件。注意：在这个地方的时候，推荐用Git Bash进行指令的操作。

    指令

    1.init指令：新建一个网站。如果没有设置 folder ，Hexo 默认在目前的文件夹建立网站。

    图中在E盘下建立一个hexo的文件夹。

    图中在E盘下成功建立一个hexo的文件夹。
    2.generate指令：生成静态文件。

    3.server指令：启动服务器。默认情况下，访问网址为： http://localhost:4000/。
